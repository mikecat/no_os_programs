OS非依存アプリケーション
========================

このフォルダには、ローダーから呼び出されるアプリケーションが格納されています。

アプリケーションのバイナリをディスクのルートに`BOOT.BIN`という名前で置くと、
ローダーによって読み込まれ、実行されます。

## 各アプリケーションの説明

### fizzbuzz.s
FizzBuzzを表示するテストです。

### primejudge.s
入力された65535までの整数の素数判定をします。

### getkey_test.s
キー入力のテストです。
入力されたキーに対応するスキャンコードとASCIIコードを表示します。  
表示は、左から順にスキャンコード、ASCIIコード、文字です。

### loader_test.s
ローダーのテストを行います。
0x7000バイト(28,672バイト)のファイルを正しく読み込めるかチェックします。

### div16.s
div命令のテストです。
各割り算は、0を押すとスキップ、0以外のキーを押すと実行できます。

### idiv16.s
idiv命令のテストです。
各割り算は、0を押すとスキップ、0以外のキーを押すと実行できます。

### div8.s
idiv命令のテストです。
各割り算は、0を押すとスキップ、0以外のキーを押すと実行できます。
