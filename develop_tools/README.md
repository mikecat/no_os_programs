OS非依存プログラム開発補助ツール
================================

開発の補助をするためのツール集です。

## 各ツールの説明

### asbl.pl
gccとobjdumpを利用し、
アセンブリ言語のソースコードを実行可能バイナリに変換するツールです。

### genimage_with_loader.pl
指定したバイナリをローダーとして使用し、
フロッピーディスクイメージを作成するツールです。

### update_loader.pl
指定したフロッピーディスクイメージのローダーを、
指定したバイナリで上書きするツールです。

### update_loader2.pl
指定したフロッピーディスクイメージのローダーを、
指定したバイナリで上書きするツールです。

update_loader.plと違い、
ファイルシステムの情報を維持したままプログラムのデータを書き込めるようになっています。

### update_loader2_hdd.pl
指定したハードディスクイメージのローダーを、
指定したバイナリで上書きするツールです。

ファイルシステムの情報を維持したままプログラムのデータを書き込めるようになっています。
また、書き込む対象のパーティションを指定することができます。
