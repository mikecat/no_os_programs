OS非依存プログラム開発補助ツール
================================

開発の補助をするためのツール集です。

## 各ツールの説明
- asbl.pl  
  gccとobjdumpを利用し、
  アセンブリ言語のソースコードを実行可能バイナリに変換するツールです。
- genimage_with_loader.pl  
  指定したバイナリをローダーとして使用し、
  フロッピーディスクイメージを作成するツールです。
- update_loader.pl  
  指定したフロッピーディスクイメージのローダーを、
  指定したバイナリで上書きするツールです。
