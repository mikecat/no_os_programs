OS非依存プログラム集
====================

OSに依存せず、BIOSなどの力を借りて「普通の」パソコンで動作するプログラム集です。
フロッピーディスクドライブとフロッピーディスクがあれば、実機でも動かせるかもしれません。  
※ある命題とその裏の真偽は必ずしも一致しません
